\## Features

\- Basic math operations (+, -, \*, /)

\- Parentheses support

\- Decimal numbers

\- Backspace and clear functions



\## Requirements

\- Java 8 or higher



\## How to Run



\### Option 1: Double click the JAR file

Simply double-click `JavaCalculator.jar`



\### Option 2: Command line

```bash

java -jar JavaCalculator.jar
